# Uso de Asistente de Código en la Práctica

Esta práctica ha sido desarrollada con la asistencia del modelo **Gemini 3.1 Pro (High)**.
El uso de la IA se centró en las siguientes fases:
1. **Ideación**: Lluvia de ideas para seleccionar la consulta más innovadora y visualmente atractiva (seleccionando la visualización espacial de cuadros de artistas españoles en museos extranjeros basada en la Práctica 6).
2. **Setup y Repositorio**: Configuración guiada paso a paso de Git y estructuración del proyecto.
3. **Desarrollo Front-End**: Asistencia en la creación de una interfaz moderna y atractiva utilizando Vanilla CSS (Glassmorphism, Dark Mode) y HTML5.
4. **Desarrollo JavaScript y SPARQL**: Adaptación de la consulta inicial a SPARQL y parseo de resultados de Wikidata utilizando jQuery y Leaflet.js de acuerdo con las buenas prácticas sugeridas en las clases.
